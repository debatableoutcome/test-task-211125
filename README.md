# Каталог пользователей

Небольшое SPA на чистых HTML, CSS и JavaScript для просмотра и управления списком пользователей с загрузкой фото и хранением в браузере.

## Возможности
- Загрузка исходных данных через `fetch` из `data/users.json`.
- Карточки с именем, возрастом, email и фото (плейсхолдер, если нет загрузки).
- Фильтр по возрасту: показывать только старше 18 лет.
- Сортировка по имени (А–Я, Я–А) и возрасту (по возрастанию/убыванию) c учетом активного фильтра.
- Загрузка фото для каждого пользователя, сохранение в `localStorage` и восстановление после перезагрузки.
- Адаптивная верстка для десктопа и мобильных устройств.

## Запуск
- Откройте `index.html` в современном браузере. Ничего устанавливать не нужно.
- Для сброса загруженных фото очистите запись `localStorage` с ключом `user-photos` для этой страницы.

## Структура
- `index.html` — разметка и блоки управления.
- `css/styles.css` — оформление и адаптивность.
- `js/app.js` — получение данных, фильтрация, сортировка, загрузка и восстановление фото.
- `data/users.json` — исходный набор пользователей.
- `screenshots/desktop.png`, `screenshots/mobile.png` — поместите сюда скриншоты для отображения в README.
